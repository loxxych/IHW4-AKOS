# AKOS-IHW4

Репозиторий содержит решения индивидуального домашнего задания по многопоточному программированию (POSIX Threads).

## Структура
- `clinic_basic/` — базовый вариант (уровень 4–5 баллов) задачи о больнице: консольная модель рабочих процессов клиники с потоками пациентов и врачей.
- `clinic_cli/` — модифицированный вариант (уровень 6–7 баллов) с вводом параметров через командную строку и генерацией случайных конфигураций.
- `clinic_io/` — расширенный вариант (уровень 8 баллов) с выводом в файл и альтернативным вводом параметров из конфигурационного файла.
